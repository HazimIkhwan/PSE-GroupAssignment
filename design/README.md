# Design

This folder details the designing process of the project, it covers the Design phase of the SDLC, where requirements are transformed into a detailed system design.

As a result, this folder contains the Use Case and Class Diagrams to convert the requirements into Unified Modelling Language to ease the process of making the UI Prototype.
